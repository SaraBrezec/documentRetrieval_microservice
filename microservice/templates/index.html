Document retrieval REST API

The Document retrieval REST API allows the user to retrieve relevant documents based on a given query.

Available methods
GET /docRetrieval/retrieval Retrieve relevant documents
POST /docRetrieval/retrieval Retrieve relevant documents (similar to the GET request)
Retrieve reeavant documents
GET /docRetrieval/retrieval

Route information

Query string parameters
query Type: String Title: Query Text Optional: false The query for which we wish to retrieve documents.
m Type: Integer Title: Number of retrieved documents Optional: true Number of retrieved documents.
Response body attributes
documents_metadata Type: List Title: Metadata of the retrieved documents Read only: true Metadata of m most relevant documents sorted descending.
documents_metadata item properties
document_source Type: String Title: Source database Read only: true Name of the document source database.
date Type: String Title: Document date Read only: true Date of the document.
title Type: String Title: Document title Read only: true Title of the document.
celex_num Type: String Title: Celex number Read only: true Celex number of the document if exists, otherwise None.
fulltextlink Type: String Title: Document link Read only: true Link to the document text.
